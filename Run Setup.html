Flavourmax Database Setup

Seeds all real master data into Google Sheets in one click

1 Paste updated script

Open your Google Sheet → Extensions → Apps Script → paste the full contents of google-sheets-setup.gs → Save.

2 Deploy as Web App

Click Deploy → Manage Deployments → Edit (pencil) → change version to New version → Deploy. Keep Execute as: Me and Access: Anyone.

3 Click Run Setup below

This page will call the web app and seed all real data automatically — companies, ingredients, suppliers, warehouses, processes and fields.
